Introduction

Ce mini-site a été créé dans le cadre du cours de TP Programmation Web

Il permet de découvrir la création de plusieurs pages HTML reliées entres elles

Objectif du mini-site

L'objectif est d'apprendre à structurer un petit projet web avec plusieurs fichiers

Pages disponibles

  • La page d'Accueil présente le contexte general du mini-site
  • La page Présentation contient les informations personnelles de l'étudiant
  • La page Contact contient les informations contact de l'étudiant
